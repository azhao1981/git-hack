# git-hack

目标是制作一个简单易用的git 工具

0. git-save git-redo git-undo git-backto They look easy to understand?  
1. remote comming son..


## Installing

All you need to do is grab the gem:

    gem install git-hack


### save
		git save "msg"

### undo
		git undo                     # 回到上一个提交

### redo
		git redo 										# 前进到下一个提交

### backto
		git backto [number]         # 跳到指定的提交
